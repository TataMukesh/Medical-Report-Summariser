# Extractive summarization of patient's medical record 



## Dependencies
* numpy
* pandas
* scipy
* nltk
* scikit-learn
* sumy


## Install
```
numpy:        $  pip install numpy
pandas:       $  pip install pandas
nltk:         $  pip install nltk
scipy:        $  pip install scipy
scikit-learn: $  pip install sklearn
summy:        $  pip install sumy


```
